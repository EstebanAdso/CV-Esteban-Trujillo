Sobre mi

Soy Carlos Esteban Trujillo Paz, de Pasto, Nariño, y soy Tecnólogo en Análisis y Desarrollo de Software (SENA). Tengo experiencia en el manejo de bases de datos como SQL Server, MySQL y MongoDB, destacándome en la creación de esquemas optimizados, consultas avanzadas y procedimientos almacenados usando Transact-SQL.

En desarrollo backend, trabajo con Java utilizando frameworks como Spring Boot para APIs RESTful, Hibernate para mapeo objeto-relacional y Jakarta EE para soluciones empresariales robustas. También tengo experiencia con Node.js, desarrollando servicios ligeros y escalables.

En el frontend, mis conocimientos abarcan JavaScript (ES6+), incluyendo promesas, async/await e iteradores, junto con HTML y CSS. Domino técnicas de diseño responsive utilizando Grid y Flexbox, asegurando interfaces modernas y adaptables.

Además, manejo herramientas como GitHub para trabajo colaborativo y Postman para pruebas de APIs. También cuento con habilidades en diseño UML, mantenimiento de sistemas operativos y hardware, y poseo un nivel de inglés conversacional para entornos internacionales.

Busco nuevas oportunidades que me permitan seguir desarrollándome profesionalmente, aportando mi experiencia, creatividad y pasión por la tecnología al éxito de su equipo. Estoy abierto a conversar y mostrar cómo puedo contribuir al crecimiento de su organización.
